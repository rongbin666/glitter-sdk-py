.. _examples:

====================
Examples
====================

For the examples on this page,
we assume you've :doc:`installed the glitter sdk Python package <install>`.


Getting Started
---------------

We begin by creating a Glitter Client from glitter_sdk to store & access data:

.. code-block:: python

    from glitter_sdk import GlitterClient
    url = 'http://127.0.0.1:26659'
    glitter_client = GlitterClient(url)

Or use the default root url:

.. code-block:: python

    from glitter_sdk import GlitterClient
    glitter_client = GlitterClient()


Create Schema
------------------------

We use JSON formatted schema to describe the data stored in Glitter.

.. code-block:: python

    schema_name="demo"
    fields = [
        {
            "name": "user_id",
            "type": "string",
            "primary": "true",
            "index": {
                "type": "keyword"
            }
        },
        {
            "name": "user_name",
            "type": "string",
            "index": {
                "type": "text"
            }
        },
        {
            "name": "email_address",
            "type": "string",
            "index": {
                "type": "text",
                "index": "false"
            }
        }
    ]
    res = glitter_client.db.create_schema(schema_name, fields)
    print(res)

If the schema is created successfully, the return would be:

.. code-block:: json

    {
        "code": 0,
        "message": "ok",
        "tx": "B88CEA8172F0B8BD7EAC3021C1B347786F74EDCD9110A7525C61237CD91FCE73",
    }

If the schema already exists, the return would be:

.. code-block:: json

    {
      "code": 500,
      "message": "schema already exist: schema_name=demo",
    }

Show Schema
------------------------

Get schema by name.

.. code-block:: python

    res = glitter_client.db.get_schema("demo")

if success:

.. code-block:: json

    {
        "code": 0,
        "message": "ok",
        "data": {
            "fields": [{
                "index": {
                    "type": "keyword"
                },
                "name": "user_id",
                "primary": "true",
                "type": "string"
            }, {
                "index": {
                    "type": "text"
                },
                "name": "user_name",
                "type": "string"
            }, {
                "index": {
                    "index": "false"
                },
                "name": "email_address",
                "type": "string"
            }],
            "name": "demo",
            "type": "record"
        }
    }

otherwise:

.. code-block:: json

    {
        "code": 505,
        "message": "SchemaNotExist",
    }

Put Document to Glitter
--------------------------------
Define a document and put it to Glitter Protocol.
For example:

.. code-block:: python

    demo_doc = {
        "user_id": "123",
        "user_name": "Bob",
        "email_address": "Bob@gmail.com"
    }
    res = glitter_client.db.put_doc(schema_name, demo_doc)

If doc is put into Glitter Protocol successfully, the return would be:

.. code-block:: json

    {
      "code": 0,
      "message": "ok",
      "tx": "49429CDC575C0ED6D021FE9BEE1D44578AC7EDAD61A25EBBF0DE72746E0064F8",
    }


Otherwise, the return may be like:

.. code-block:: json

    {
      "code": 500,
      "message": "RPC error -32603 - Internal error: tx already exists in cache",
    }


Check Existence of Document
-----------------------------------
Query by primary key of the document.

.. code-block:: python

    schema_name = "demo"
    user_id = "123"
    res = glitter_client.db.get_docs(schema_name, [user_id])

We will have a document return as below:

.. code-block:: json

    {
        "code": 0,
        "message": "ok",
        "data": {
            "total": 1,
            "hits": {
                "123": {
                    "_schema_name": "demo",
                    "user_id": "123",
                    "user_name": "Bob",
                    "email_address": "Bob@gmail.com"
                }
            }
        }
    }


Search in the Schema
-------------------------------------------------
This search is the standard query for performing a full-text search, including options for fuzzy matching.

.. code-block:: python

    query_word = "Bob"
    query_field = ["user_name"]
    res = glitter_client.db.search(schema_name, query_word, query_field)

The hit result like:

.. code-block:: json
